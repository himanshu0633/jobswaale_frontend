Jobseeker portal routes and page structure

This folder contains the Jobseeker portal pages and layout.
- `JobseekerLayout.jsx`: shared jobseeker layout with header, sidebar, footer and auth guard.
- `dashboard/JobseekerDashboard.jsx`: the main jobseeker dashboard page.
- `applications/JobseekerApplications.jsx`: job applications list.
- `interviews/JobseekerInterviews.jsx`: interview schedule page.
- `reports/JobseekerReports.jsx`: jobseeker reports page.
- `selected/JobseekerSelected.jsx`: selected job list page.
